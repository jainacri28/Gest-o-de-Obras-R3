# Gestão de Obras

Aplicativo em Streamlit para cadastro, acompanhamento e monitoramento completo de obras.

## Recursos

- dashboard gerencial com indicadores e alertas;
- cadastro de clientes e obras;
- etapas da obra com progresso, observações e custos;
- mão de obra por etapa;
- cronograma e linha do tempo;
- controle de custos previstos x reais;
- dados de exemplo com 10 clientes e obras fictícias.

## Como executar

1. Instale as dependências:

```bash
pip install -r requirements.txt
```

2. Rode o aplicativo:

```bash
streamlit run app.py
```
